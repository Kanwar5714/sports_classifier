Objective

The core goal of this project is to simplify sports object recognition through automation. Whether it's a football, hockey puck, or cricket ball, this system can detect and classify the object accurately, which can assist in educational tools, training software, or retail product management.

Technology Stack

  • Backend: Flask (Python)
  • Machine Learning: TensorFlow + Keras
  • Frontend: HTML5, Bootstrap 5
  • Deployment: Render (Cloud Hosting)

Dataset

The model was trained on a diverse dataset of 15 sports object classes including volleyballs, table tennis balls, rugby balls, and more. Images were normalized, resized to 224x224, and augmented to improve generalization. The dataset was carefully curated to ensure balanced class representation.

Features

  • Upload images in real-time and get instant predictions.
  • Displays the uploaded image along with the prediction.
  • User-friendly and mobile-responsive UI.

Future Enhancements
